## Acessing Neo4J Database:

- Go to neo4j.com 
- Click `Get Started`
- Scroll down to `AuraDB Free`
    - Click Start Free
    - This will take you to the Neo4J AuraDB Console
- Connect to the `Chicago Roadways` instance with the credentials in `neo4j_demo/creds.txt`
